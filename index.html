About Riot Fest Schedule

🎵 Festival Schedule PWA V1.4

A modern Progressive Web App for the Riot Fest 3-day concert schedule. Built with offline functionality, real-time updates, and a beautiful mobile-first design.

✨ Features

  • 📱 PWA Support - Install on your device
  • 🔴 Now Playing - See what's currently live
  • ⭐ Favorites - Star your favorite artists
  • 🎭 Stage Filters - Filter by stage or favorites
  • 📱 Offline Access - Works without internet
  • ⚡ Real-time Updates - Always current
